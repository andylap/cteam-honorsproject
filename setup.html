Setup

Overview

This page will demonstrate how to setup:

  • Git on macOS, Linux, and Windows
  • C++ on macOS, Linux, and Windows
  • Java on macOS, Linux, and Windows

Installing Git

macOS

Git is probably already installed into your system if you are running macOS. You can check using the following command in your terminal: 

$ git --version

If it is not installed, just install Homebrew and enter the following command: 
$ brew install git

Linux

Like macOS, Git is probably already installed into your system if you are running Linux. It is definitely installed if you are Purdue's ecngrid. You can check using the following command in your terminal: 

$ git --version

If it is not installed, just enter the following commands: 
$ sudo apt-get update 
$ sudo apt-get install git

Windows
  1. The installation for Windows is a bit different than with Linux and macOS. First you're going to download the latest version of Git for Windows Installer.
  2. Once the installer is started follow the wizzard screen prompts through Next and Finish with the default options.
  3. Open a Command Prompt (or Git as if during installation you elected not to use Git from the Windows Command Prompt).
  4. Run the following command to configure your Git username and email: 
    $ git config --global user.name "your name here" $ git config --global user.email "your git email here"

C++ Setup

When getting set up to compile a C++ program, it is important to pick the right compiler for your specific operating system. See our more in-depth description of C++ compilers here.

Windows

If you are working on Windows there are few extra step you must take to get your environment setup with C++ once you've picked your compiler. The following steps apply to the GCC MinGW installation method with Windows.
  • When installing MinGW you must at least install the gcc-core, gcc-g++, binutils, and the MINGW runtime, but feel free to install more
  • You need to add the bin subdirectory on the MinGW folder you just installed to your PATH environment variables so that you can use these tools on the command line
    • Navigate to the environment variable settings.
    • Edit the Path variable in your system environment variables.
    • Add the path to the bin subdirectory of MinGW to the Path variable.
  • Now you will be able to use gcc, g++ and other GNU tools in the Windows command line

Installing Java

Installing Java is the same on macOS, the most common flavors of Linux (Ubuntu, Debian, etc.), and Windows.

  1. Download the Java Development Kit (JDK) through Oracle's website. Once you're there, click on JDK Download and choose the download for your OS:
    • For macOS, choose the macOS Installer.
    • For a Linux flavor like Debian and Ubuntu, choose the Linux x64 Debian Package if you're using an Intel- or AMD-based processor (which is the most common case).
    • For Windows, choose the Windows x64 Installer.
  2. Open the installer or package and follow the instructions in the setup.
  3. When setup is complete, you can check that it has installed successfully by using the following command in your terminal (Command Prompt if on Windows): 
    java -version

Choosing an IDE

There are several options for you to choose when it comes to an Integrated Development Environment (IDE). IDEs are incredibly useful when developing Java programs since it includes both a space for you to code your program as well as provide powerful tools for debugging.

Eclipse is the most popular IDE for Java developers, which includes a fully-integrated environment that is adaptible to different needs and uses. It includes a solid collection of libraries, as well as a large library of plugins available to suit your needs. Eclipse is available to download through the Eclipse Foundation's website at eclipse.org.

Download Eclipse

IntelliJ IDEA offers features that are geared towards more advanced developers, while maintaining a simple user interface for beginners. It offers smart completion and chain completion when coding along with powerful debugging tools and Git support. IntelliJ IDEA has a free Community Edition, as well as an Ultimate Edition available to download through JetBrains's website at jetbrains.com. However, as a student, you are able to download the Ultimate Edition for free if you apply for an education license.

Download IntelliJ IDEA

NetBeans is the oldest of the top IDEs for Java (released 1997), which was the official IDE for Java 8 when released in 2014. It features visual debugging for GUI applications as well as extensions for other languages like C/C++. NetBeans is available to download through the Apache website at netbeans.apache.org.

Download Apache NetBeans

Visual Studio Code is the newest source-code editor with features that make it act like an IDE. By installing the Java Extension Pack, you are able to use Microsoft's IntelliSense code completion feature along with direct Git support. Visual Studio Code is available to download for free on Microsoft's website.

Download Visual Studio Code
